## 今後の利用方法

### ✅ 新しい作業開始時
```bash
mkdir -p 作業履歴/YYYYMMDD_プロジェクト名
# その中に設計・実装・テスト結果を格納
```

### ✅ Git管理
```bash
git add .
git commit -m "【プロジェクト】作業内容説明"
```

### ✅ ファイル配置ルール
- **設計資料・参照資料** → `参考資料/`
