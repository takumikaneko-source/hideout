# 参考資料

外部から受領した資料・案件の設計書・仕様書など「参照する側」の資料を案件別に管理する。

## フォルダ構成

```
参考資料/
├── 矢崎案件/     # taxi-cloud 関連の受領資料・仕様書
└── （案件名）/   # 新規案件ごとにフォルダを追加
```

## 配置ルール

- **自分が作った成果物は置かない**（それは 作業履歴/ へ）
- **使い回せる知見に加工したものも置かない**（それは 知識ベース/ へ）
- ここに置くのは「受け取った資料」「参照する原本」のみ
- 案件フォルダ内の整理は自由（資料の種類・時期など、案件の実情に合わせる）

## 注意

- 受領資料に機密情報（認証情報・個人情報）が含まれる場合は格納前に除外・マスクする
- 出典（誰から・いつ・何のために受領したか）が わかるよう、必要ならフォルダ内にメモを添える
