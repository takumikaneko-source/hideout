# hideout — クロスプロジェクト知識庫

システムエンジニアの作業実績・参考資料・ナレッジを一元管理するリポジトリ。

## 📂 ディレクトリ構成

```
hideout/
├── 00_共通/              # 共通資産・テンプレート
├── 02_資料/              # 設計資料・仕様書
├── 04_作業履歴/          # プロジェクト別実装履歴（taxi-cloud他）
├── 08_knowledges/        # 横断的ナレッジベース
└── templates/            # 資料作成用テンプレート
```

## 🎯 用途

- **taxi-cloud** および他プロジェクトの設計・実装・運用知識を集約
- 参照資料・手順書・設計説明の保管
- チーム間・プロジェクト間の知識共有
- 過去実績の検索・再利用

## ⚠️ 注意

- **機密情報は含まない**（AWS認証情報・個人情報は除外）
- Google Sheets（.gsheet）ファイルはバイナリのままで、ローカルでは参照になります
- Git で管理しているため、commit メッセージは作業内容を明記してください

## 📝 更新ルール

- 資料の大きな更新・新規ナレッジ追加時は commit を切る
- 個人名・役職名などの特定情報は避ける（「運用チーム」等で一般化）
- 日本語のみ使用
