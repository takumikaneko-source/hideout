# テンプレート

資料作成・記録用の定型ファイルと共通素材を管理する。

## テンプレート一覧

| ファイル | 用途 | 使いどころ |
|----------|------|------------|
| [作業記録.md](作業記録.md) | 作業履歴フォルダの README.md | 新規作業の着手時 |
| [ナレッジ記事.md](ナレッジ記事.md) | 知識ベースの記事 | 使い回せる知見を得たとき |
| [手順書.md](手順書.md) | 操作・運用手順書 | 他者に渡す手順を書くとき |
| [HTML資料スケルトン.html](HTML資料スケルトン.html) | 閲覧用・説明用のHTML資料 | 設計説明・報告などの見せる資料 |

## 使い方

テンプレートをコピーして目的の場所に配置し、`{{ }}` の箇所を埋める。

```
例: 新規作業の開始
テンプレート/作業記録.md → 作業履歴/20260710_○○対応/README.md
```

## 共通素材

- `アイコン/` — 資料用アイコン画像

## テンプレート自体の更新

- 使ってみて足りない項目・不要な項目があれば、テンプレート自体を直して commit する
- 大きく変えるときは既存の記録を壊さないよう、項目の削除より追加を優先する
